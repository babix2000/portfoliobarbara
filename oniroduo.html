01. SOBRE

Oniroduo é uma marca nacional de roupa urbana, onde o objetivo de cada coleção que trazem é garantir o conforto e qualidade, sendo que o fabrico das suas peças é 100% português.

É um projeto que tive a oportunidade de o desenvolver desde raíz, trabalhando tanto na parte de frontend como de backend.

Foi com este projeto que iniciei a minha jornada de aprendizagem no universo das vendas online e adentrei no mundo do WooCommerce, aprimorando as minhas habilidades nessa área.

Cada desafio enfrentado e cada solução implementada representaram oportunidades valiosas de crescimento, capacitando-me a oferecer um serviço ainda mais completo e eficiente para a Oniroduo e os seus clientes.

02. DESENVOLVIMENTO DO SITE

- BACKEND
PHP WORDPRESS WOOOCOMMERCE WP BAKERY BUILDER

Com o WordPress como backoffice e o WooCommerce como ferramenta de comércio eletrónico, conseguimos integrar perfeitamente o conteúdo e a funcionalidade da loja.

Além de fornecer templates e funcionalidades de compra/checkout e registo de conta de usuário pré-definidas, o WooCommerce também desempenha um papel fundamental na gestão de encomendas e estratégias de marketing da loja, onde proporciona recursos essenciais para impulsionar o crescimento do negócio, como promoções, descontos e análises de desempenho.

As páginas foram construídas utilizando o plugin WP Bakery Builder, uma ferramenta que adota uma abordagem baseada em blocos para a criação de layouts. Essa metodologia revelou-se extremamente prática, eficiente e de fácil compreensão, tornando o processo de desenvolvimento mais ágil e intuitivo. Além disso, o WP Bakery Builder oferece a vantagem de proporcionar autonomia ao usuário, permitindo que ele crie páginas simples por conta própria, graças à sua interface amigável e responsiva.

Para alcançar a personalização desejada e alinhar a loja com o design original, foi necessário recorrer a filtros e ações para adicionar ou remover conteúdo nas páginas template do WooCommerce. Esse processo demandou o desenvolvimento de funções e shortcodes, envolvendo um trabalho extenso com PHP. Essa abordagem permitiu ajustar detalhes específicos da apresentação de produtos, layouts de página e elementos de navegação.

- FRONTEND
HTML CSS JQUERY SLICK.JS

Além de utilizar HTML para a estrutura e CSS para estilização do site, também integrei JQuery para ajudar nalgumas transições, animações e eventos. Assim como no desenvolvimento de funcionalidades de filtragem de produtos na página da loja, garantindo que os clientes pudessem navegar facilmente pela variedade de produtos oferecidos. Essa abordagem não só melhorou a usabilidade do site, mas também contribuiu para uma experiência de compra mais eficiente e agradável.

O Jquery foi especialmente crucial para manipular elementos nas páginas geradas pelo WooCommerce, uma vez que o design original do site não correspondia ao template padrão fornecido pelo plugin. Dado que não é recomendado modificar diretamente os arquivos originais do WooCommerce, recorri ao JQuery para essa personalização necessária, garantindo uma experiência de usuário fluída e coesa.

Para criar o slide inicial na homepage, contei com a assistência de uma biblioteca chamada Slick.js. Essa ferramenta ofereceu uma solução robusta e flexível para implementar o carrossel de imagens desejado, permitindo uma navegação suave e uma apresentação visualmente atraente. A utilização dessa biblioteca facilitou significativamente o processo de desenvolvimento, garantindo um resultado final profissional e intuitivo para os visitantes do site.

- OUTROS PLUGINS USADOS
PODS CONTACT FORM 7 WP MAIL SMTP TI WISHLIST WPML

PODS: Para a ajuda na criação de novos post types e custom fields. Essa ferramenta facilita significativamente a implementação e o gerenciamento de campos personalizados, oferecendo uma interface intuitiva e flexível para o desenvolvedor. Além disso, a capacidade de trabalhar com os valores dos campos diretamente no código PHP proporciona uma integração perfeita e um controle total sobre o conteúdo do site.

Contact Form 7: Pude facilmente criar formulários e personalizar os campos necessários para a Newsletter e a página de contatos. A sua integração simples e interface intuitiva tornaram o processo de implementação dos formulários uma tarefa rápida e fácil.

WPML: Para a ajuda na tradução de todo o conteúdo no site para as outras línguas, o Wordpress Multilingual é uma excelente ferramenta para isso, sendo que é altamente compatível com uma ampla gama de temas e plugins do WordPress. Ele simplifica o processo de tradução, oferecendo uma interface amigável para gerenciar todo o conteúdo traduzido num único painel.

TI Wishlist: Permite que os clientes salvem produtos desejados para uma lista de desejos personalizada, facilitando o retorno à loja e a conclusão da compra posteriormente. De fácil implementação e personalização, possibilitando a integração dos seus recursos por meio de shortcodes em código PHP.

WP Mail SMTP: Para garantir a entrega de emails desenvolvidos pelo Contact Form 7 ou mesmo por meio de código PHP, este plugin é uma ferramenta essencial para isso, pois permite o redirecionamento de envio de e-mails através de um serviço de e-mail SMTP. A configuração dele é fácil, uma vez que oferecem um tutorial para cada tipo de protocolo.
